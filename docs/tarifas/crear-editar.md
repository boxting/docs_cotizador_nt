# Crear y Editar Tarifa

Al registrar una tarifa se solicita la siguiente informacion:

- Variables comerciales y logisticas (cliente, producto, terminales, zona, etc.).
- Precio.
- Vigencia: fecha de inicio y fecha de fin.
